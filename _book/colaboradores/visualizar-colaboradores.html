Visualizar todos os colaboradores

[GET] https://app.wecarecorp.com.br/api/v1/users


Esta ação não exige nenhum parâmetro.


Cabeçalhos:

Cabeçalho Valor
Authorization Basic Token-de-Autenticação (Autenticação a partir do par api_key:secret)

Resposta bem sucedida:

Código: 200 OK

Exemplo de requisição:

var settings = {
  "async": true,
  "crossDomain": true,
  "url": "https://app.wecarecorp.com.br/api/v1/users",
  "method": "GET",
  "headers": {
    "authorization": "Basic Token-de-Autenticacao",
    "cache-control": "no-cache"
  }
}

$.ajax(settings).done(function (response) {
  console.log(response);
});
