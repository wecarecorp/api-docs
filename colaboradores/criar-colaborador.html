Criar um colaborador

[POST] https://app.wecarecorp.com.br/api/v1/users


Esta ação não exige nenhum parâmetro.


Cabeçalhos:

Cabeçalho Valor
Authorization Basic Token-de-Autenticação (Autenticação a partir do par api_key:secret)
Content-Type application/json


Corpo da requisição:

{
  "user": {
    "name": nome (string),
    "email": email (string),
    "sector": setor (string, opcional),
    "function": cargo (string, opcional),
    "cpf": cpf (string),
    "alternative_login": login alternativo (string, opcional)
  }
}

Código: 201 Created

var settings = {

    "content-type": "application/json",
