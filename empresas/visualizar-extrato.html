Visualizar extrato da empresa

[GET] https://app.wecarecorp.com.br/api/v1/transactions


Parâmetros:

Parâmetro Valores aceitos Descrição
month (inteiro, optativo) 1-12 Mês das transações
order_by (string, optativo) created_by Ordena resultados por data de criação
reverse (boolean, optativo) true, false Reverte um resultado ordenado
created_min (string, optativo) data (dd-mm-aaaa) Data de criação mínima
created_max (string, optativo) data (dd-mm-aaaa) Data de criação máxima
max_results (inteiro, optativo) Qualquer inteiro Limite de resultados


Cabeçalhos:

Cabeçalho Valor
Authorization Basic Token-de-Autenticação (Autenticação a partir do par api_key:secret)

Resposta bem sucedida:

Código: 200 OK

var settings = {
  "async": true,
  "crossDomain": true,
  "url": "https://app.wecarecorp.com.br/api/v1/transactions",
  "method": "GET",
  "headers": {
    "authorization": "Basic Token-de-Autenticacao",
    "cache-control": "no-cache"
  }
}
